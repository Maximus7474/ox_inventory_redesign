# Installation Guide

## Requirements
- Base Ox inventory requirements ()
- Illenium Appearance - [github repository](https://github.com/iLLeniumStudios/illenium-appearance)

## Installation Steps
1. Download the [latest release](https://github.com/Maximus7474/ox_inventory_redesign/releases/latest/download/ox_inventory.zip) or clone the repository
  - **If cloning the repository** or downloading via the Code button, requires **you** to build the UI.
    You'll need [`bun`](https://bun.com), navigate to `web`, run `bun install` and then `bun run build`

2. Download & install illenium appearance on your server
3. Drag'n drop the `illenium-appearance` folder within this repository (`INSTALLATION/illenium-appearance`) in your illenium appearance resource.
   This will add / update code of the resource to make it compatible with this redesign

> [!NOTE]
> If these steps are too hard, or if you can't make it work then try again.
